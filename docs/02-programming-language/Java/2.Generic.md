# Generic `<T>` demo
```java
public class BoxDemo {
    public static void main(String[] args) {
        Box<String> strBox = new Box<>();
        strBox.set("Hello 123");
        System.out.println(strBox.get());

        Box<Integer> intBox = new Box<>();
        intBox.set(123);
        System.out.println(intBox.get());
    }
}

class Box<T> {
    private T value;

    public void set(T v) {
        this.value = v;
    }

    public T get() {
        return value;
    }
}
```

Output:
```
Hello 123
123
```
